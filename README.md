# thesis

Research repository for thesis.

## Quick Start

### Build Documentation Locally
```bash
# Start development server
jupyter-book start

# Build static HTML (for deployment)
jupyter-book build --html
```

### Deploy to GitHub Pages
Push to `main` branch and GitHub Actions will automatically deploy.

## Structure

- `data/`: Raw and processed data
- `figures/`: Source files and exported figures
- `notebooks/`: Jupyter notebooks for exploration and analysis
- `scripts/`: Python, C++, and bash scripts
- `notes/`: Literature notes and concept documentation
- `reports/`: Weekly and daily progress reports
- `paper/`: LaTeX manuscript
